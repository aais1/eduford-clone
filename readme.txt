## About

The Eduford University Website Clone is a simplified version of the original Eduford University website. It was created as part of a learning exercise to practice web development skills and gain hands-on experience with HTML, CSS, and potentially JavaScript.

## Features

- Home page showcasing the university's main features and highlights.
- Facilities section providing an overview of the university's amenities.
- Degree programs section with detailed descriptions of available degrees.
- Responsive design to ensure the website looks great on various devices.
